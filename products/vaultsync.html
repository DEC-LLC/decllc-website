What VaultSync Does

Complete backup, restore, and cross-platform sync for virtualized and physical infrastructure. No vendor lock-in. No agents required.

Multi-Hypervisor Backup

Back up VMs on ESXi, KVM/libvirt, Proxmox VE, Hyper-V, and VirtualBox from a single orchestrator. No per-hypervisor tools needed.

Physical Server Backup

Back up bare-metal Linux and Windows servers with LVM snapshots, btrfs, VSS, or live rsync. Full disk images for P2V conversion.

Cross-Hypervisor Sync

Continuously sync VMs between ESXi, KVM, and Proxmox with automatic format conversion. Fail over from VMware to free Proxmox in minutes.

CBT & Dirty Bitmap Incrementals

Changed Block Tracking for ESXi and dirty bitmaps for KVM. Only changed blocks are transferred — 100x faster than full backups.

Cloud Storage Targets

Back up to AWS S3, Oracle Cloud, Azure Blob, or Google Cloud Storage. Client-side AES-256-GCM encryption — your cloud provider never sees your data.

Restore Anywhere

Restore to the same hypervisor, a different hypervisor, a physical server, or the cloud. One backup, unlimited restore targets.

Hypervisor Host Backup

Back up ESXi, Proxmox, and KVM host configurations. Recover an entire hypervisor from bare metal in minutes.

Web Dashboard

How It Works

VaultSync is a control-plane orchestrator. It sends SSH commands to source and target systems — backup data never touches the orchestrator. Runs on a 2GB VM.

1. Configure

Define your hypervisors, VMs, backup targets, and schedules in a single YAML file.

2. Backup

VaultSync snapshots VMs, extracts changed blocks (CBT/dirty bitmap), and streams data directly to your backup target.

3. Restore or Sync

Restore to any platform with automatic format conversion. Or set up continuous sync for disaster recovery.
